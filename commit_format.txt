Github Commits Message Convention

Types of commits
Feat:		The new feature you are adding to a particular application
Fix:		A bug fix
Style:		Feature and updates related to styling (Art)
Refactor:	Refactoring a specific section 
Merge:		Everything related to merging
Docs:		Everything related to documentation
Chore:		Regular code maintenance	

(Having multiple types in one commit)
Summary: Short description of commit

Description:
[*Type*]
- Content

[*Type*]
- Content

(A commit with one type)
Summary: [*Type*] Short description of commit

Description:
- Content

(Example - A commit with one type)
Summary: [Fix] Fix a Rendering Bug

Description:
- Fix issue where two objects were overlapping
